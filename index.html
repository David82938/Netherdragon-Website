How to join

Minecraft Java Edition

  1. Open Minecraft and select "Multiplayer"
  2. Click "Add Server"
  3. Enter the server IP: play.netherdragon.net
  4. Click "Done"
  5. Select "Netherdragon Network" from your server list and click "Join Server"

Minecraft Bedrock Edition

  1. Open Minecraft and select "Play"
  2. Go to the "Servers" tab
  3. Scroll down and click "Add Server"
  4. Enter the server name: Netherdragon Network
  5. Enter the server IP: play.netherdragon.net
  6. Enter the server Port: 19132
  7. Click "Save"
  8. Select "Netherdragon Network" from your server list and click "Join Server"
